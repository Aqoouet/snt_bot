# SNT Balance Bot

Telegram-бот для учёта баланса СНТ. Написан на Go без внешних зависимостей.

## Возможности

- Регистрация **расходов** (описание → сумма)
- Регистрация **приходов** (описание → сумма)
- Просмотр **текущего баланса**
- **Выгрузка в Excel** (.xlsx) с историей операций
- **Авторизация пользователей** через администратора
- Хранение данных в локальном `data.json`

## Команды

| Команда | Кто | Действие |
|---------|-----|----------|
| `/start` | все | приветствие |
| `/expense` | авторизованные | зарегистрировать расход |
| `/income` | авторизованные | зарегистрировать приход |
| `/balance` | авторизованные | текущий баланс |
| `/export` | авторизованные | выгрузка в Excel |
| `/cancel` | авторизованные | отменить текущее действие |
| `/adduser <id>` | admin | добавить пользователя |
| `/removeuser <id>` | admin | удалить пользователя |
| `/users` | admin | список авторизованных |
| `/pending` | admin | список запросов на доступ |

## Установка и запуск

### 1. Создайте бота

Получите токен у [@BotFather](https://t.me/BotFather).

Узнайте свой Telegram ID (например через [@userinfobot](https://t.me/userinfobot)).

### 2. Конфигурация

Скопируйте пример конфига:

```bash
cp config.example.json config.json
```

Отредактируйте `config.json`:

```json
{
  "bot_token": "ВАШ_ТОКЕН",
  "admin_id": ВАШ_TELEGRAM_ID,
  "starting_balance": 50000.00,
  "data_file": "data.json"
}
```

Или используйте переменные окружения:

```bash
export BOT_TOKEN=ВАШ_ТОКЕН
export ADMIN_ID=123456789
export STARTING_BALANCE=50000
```

### 3. Сборка и запуск

```bash
go build -o snt-bot .
./snt-bot
```

### 4. Авторизация пользователей

1. Пользователь пишет любое сообщение боту
2. Администратор получает уведомление с ID пользователя
3. Администратор выполняет `/adduser <id>`
4. Пользователь получает подтверждение и может работать

## Структура файлов

```
snt-bot/
├── main.go          # точка входа, загрузка конфига
├── bot.go           # Telegram API (long polling, sendMessage, sendDocument)
├── storage.go       # хранение данных в data.json
├── handlers.go      # обработчики команд и FSM-диалогов
├── excel.go         # генерация .xlsx без внешних зависимостей
├── go.mod
├── config.example.json
└── data.json        # создаётся автоматически
```
